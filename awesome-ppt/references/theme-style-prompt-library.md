## More Advanced "Style-only" Prompts

These prompts are useful when you already have the slide content and only want to redesign the visual style.

### 1. Clean Business Redesign

```text
Redesign this PowerPoint into a clean, modern business presentation.
Use executive-style layouts, strong slide titles, concise text, white space, and professional charts.
Keep the content accurate, but improve visual hierarchy, spacing, typography, and consistency.
The deck should look like a polished consulting presentation.
```

### 2. Premium Dark Mode Redesign

```text
Redesign this PowerPoint using a premium dark mode style.
Use charcoal or deep navy backgrounds, subtle gradients, glassmorphism cards, neon blue accents, and clean sans-serif typography.
Make charts and diagrams visually clear with high contrast.
The final deck should feel modern, technological, and high-end.
```

### 3. Academic Formal Redesign

```text
Redesign this PowerPoint into a formal academic presentation.
Use a clean scholarly layout, restrained colors, readable fonts, structured section pages, and clear diagrams.
Keep the tone rigorous and suitable for a university or research conference.
Avoid excessive decoration and make the logic of the argument easy to follow.
```

### 4. Creative Visual Redesign

```text
Redesign this PowerPoint into a creative, visually engaging presentation.
Use bold typography, modern composition, asymmetrical layouts, tasteful colors, and memorable visual metaphors.
Make the slides attractive without reducing readability.
The deck should feel fresh, stylish, and suitable for a creative classroom or brand presentation.
```

### 5. Data Storytelling Redesign

```text
Redesign this PowerPoint as a data storytelling deck.
Turn tables and dense numbers into clean charts, KPI cards, comparison visuals, and annotated insights.
Each slide should answer one key question and highlight the main takeaway.
Use a modern dashboard-inspired style with clear hierarchy and minimal clutter.
```
